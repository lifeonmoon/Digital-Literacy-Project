# Task 1 — Digital Literacy Infographic

**Tool Used:** Canva (canva.com)

## Description
A single-page infographic titled *"The Digital Student: A Quick Guide to Digital Literacy"*, designed using Canva's free infographic templates.

## Topics Covered (4 out of 5)
1. **What is Digital Literacy?** — More than just using tech; it is the ability to find, evaluate, create, and safely communicate information across digital platforms.
2. **Useful Digital Tools for Students** — Google Workspace (Docs, Sheets, Forms) for collaboration, GitHub for version control & portfolios, and communication platforms (Discord/Slack/Teams).
3. **Safe Internet Practices** — Using strong passwords + 2FA, verifying links before clicking (phishing awareness), and protecting personal information online.
4. **Professional Online Presence** — Maintaining an active, clean LinkedIn profile to showcase academic journey and connect with mentors.
